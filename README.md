# Python Foundations Assignment

## 📌 Overview
This project contains solutions for **Python Foundations tasks** across three levels:

- **Level 1 (Basic Problems)**
- **Level 2 (Intermediate Problems)**
- **Level 3 (Advanced CLI Applications)**

Each task is implemented using clean, readable, and well-structured Python code following best practices.

---

## 🧩 Level 1 Tasks

### 2. Factorial Calculator (`Level 1 Factorial.py`)
Computes the factorial of a number using a loop.

### 3. String Reverser (`Level 1 String Reverse.py`)
Reverses a given string.

---

## 🔢 Level 2 Tasks

### 1. Largest Number Finder (`level2_largest.py`)
Finds the largest number from a list of user inputs.

### 3. Fibonacci Generator (`Level 2 Fibonacci.py`)
Generates a Fibonacci sequence up to a specified number of terms.

---

## 🧮 Level 3 Tasks

### 1. CLI Calculator (`Level 3 Calculator.py`)
A command-line calculator that supports:
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

Includes input validation and error handling.
